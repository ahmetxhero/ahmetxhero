OynaXo, klasik 3 Taş oyununu mobil cihazlara taşıyan eğlenceli, hızlı ve stratejik bir mobil oyundur. Geleneksel zeka oyunu deneyimini modern tasarımıyla birleştirerek hem nostalji hissi yaşatır hem de çağdaş bir oyun keyfi sunar.

## 🇬🇧 English

OynaXo is a user-friendly Tic-Tac-Toe game developed with modern web technologies. You can play against an AI opponent or play online matches with your friends.

### 🌐 Live Demo

Play now: [OynaXo Web Application](https://oynaxo.web.app)

### ✨ Features

- 🤖 AI-powered opponent
- 👥 Two-player mode
- 🎯 Easy, medium, and hard difficulty levels
- 📱 Responsive design (mobile-friendly)
- 🌙 Dark/Light theme support
- 📊 Game statistics
- 🏆 Scoreboard
- 🌍 Multi-language support (Turkish, English, French)

### 🛠️ Technologies Used

- React.js
- Firebase
- Tailwind CSS
- TypeScript
- Vite
- i18next (Multi-language support)

### 🚀 Installation

To run the project locally:

1. Clone the repository:
```bash
git clone https://github.com/oynaxo/oynaxo.git
```

2. Navigate to the project directory:
```bash
cd oynaxo
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm run dev
```

### 🎮 How to Play?

1. Visit [oynaxo.web.app](https://oynaxo.web.app) to start playing
2. Choose game mode (Single player or Two player)
3. Select difficulty level in single player mode
4. Take turns placing X and O marks
5. Win by getting three of your marks in a row (horizontally, vertically, or diagonally)!
