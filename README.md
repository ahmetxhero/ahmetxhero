## 🎮 Oyun Modları

### 🤖 Tek Oyunculu
- Yapay zeka ile karşılaşma
- 3 farklı zorluk seviyesi
- İstatistik takibi
- Başarı rozetleri

### 👥 İki Oyunculu
- Arkadaşlarınla çevrimiçi oyna
- Aynı cihazda pass & play
- Özel oda oluşturma
- Arkadaş listesi

### 🏆 Turnuva Modu
- Günlük turnuvalar
- Özel ödüller
- Liderlik tablosu
- Sezonluk yarışmalar
